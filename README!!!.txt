======== การใช้งาน ========
-กด run.bat เพื่อใช้งาน 

Date
-ปรับแต่งวันที่ ได้ที่ Enable Date Selection และเลือก "ปี เดือน วัน" ได้เลย
-!!SET ย้อนหลังได้ 5 ปี!!

News
-ปรับแต่งจำนวนข่าวที่ต้องการให้อ่าน ได้ที่ Enable Card Amount Set ใส่เลขในช่อง Card Amount 
 โปรแกรมจะอ่านข่าวในแต่ละบริษัทตามตัวเลข

Keywords
-โปรดใส่ Keywords และแยกแต่ละ Keywords ด้วย comma "," เช่น  ดี, เลิก  เป็นต้น

Start
-หลังจากนั้นกด Start เพื่อเริ่มทำงาน ด้านล่างจะมี Log บอกการทำงานของโปรแกรม
 หลังจากทำงานเสร็จ โปรแกรมจะนำข้อมูลเปลี่ยนเป็นไฟล์ Excel และใส่ใน Folder ชื่อ reports
 ชื่อ File จะประกอบด้วย ปี-เดือน-วัน_ชั่วโมง-นาที-วืนาที_CompanyData.xlsx

Stop
-หากต้องการหยุดให้กดปุ่ม Stop หลังจากนั้นโปรแกรมจะหยุดการทำงาน และ นำข้อมูลเปลี่ยนเป็นไฟล์ Excel และใส่ใน Folder ชื่อ reports

======== ข้อควรระวัง ========
-โปรดอย่าให้หน้าจอกดับขณะโปรแกรมทำงาน

======== หากเกิดปัญหา ========
-หากต้องการแก้ไข Code ด้วยตนเอง โปรดแก้ไขที่ "SuperScraper.py"

-โปรดติดต่อหากต้องการให้แก้ไป ข้อผิดพลาด

"โปรแกรมยังอยู่ในช่วงพัฒนา"
